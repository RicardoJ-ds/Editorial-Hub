The PRD compliance tracker lives at `tasks/prd-compliance-tracker.md`. It maps every requirement from the 3 PRDs to implementation status (DONE/PARTIAL/MISSING/DEFERRED) with notes on why things are missing and fix priority (P0/P1/P2).

Must be updated whenever:
- A PRD gap is fixed
- A new feature is added
- Coverage percentages change

Also referenced in `CLAUDE.md` at repo root under "PRD Compliance" section.
