# 90 · Archive — resolved / stale

Where files go when they're no longer current but worth keeping for traceability. **Don't
delete history — move it here.**

On a lint pass, a file marked `> ⚠️ STALE as of YYYY-MM-DD — <reason>` that's fully resolved
(view shipped, question answered, decision superseded) moves into this folder. Leave the
STALE banner on it so anyone who finds it knows why it's here and what replaced it.
